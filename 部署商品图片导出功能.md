### 1. 后端部署

#### 1.1 添加Maven依赖

在 `meituan-backend/pom.xml` 的 `<dependencies>` 标签内添加：

```xml
<!-- 图片处理库 -->
<dependency>
    <groupId>net.coobird</groupId>
    <artifactId>thumbnailator</artifactId>
    <version>0.4.19</version>
</dependency>

<!-- 阿里云OSS SDK（如果还没有） -->
<dependency>
    <groupId>com.aliyun.oss</groupId>
    <artifactId>aliyun-sdk-oss</artifactId>
    <version>3.17.1</version>
</dependency>
```

#### 1.2 配置阿里云OSS

在 `meituan-backend/src/main/resources/application.yml` 中添加：

```yaml
# 阿里云OSS配置
aliyun:
  oss:
    endpoint: oss-cn-hangzhou.aliyuncs.com  # 替换为你的OSS地域
    access-key-id: YOUR_ACCESS_KEY_ID       # 替换为你的AccessKeyId
    access-key-secret: YOUR_ACCESS_KEY_SECRET  # 替换为你的AccessKeySecret
    bucket-name: your-bucket-name           # 替换为你的Bucket名称
    base-url: https://your-bucket-name.oss-cn-hangzhou.aliyuncs.com  # 访问URL前缀
```

**获取OSS配置信息**：
1. 登录阿里云控制台
2. 进入OSS服务
3. 创建Bucket（如果还没有）
4. 获取AccessKey（在用户中心）
5. 记录Endpoint和Bucket名称

### 2. 前端部署

#### 2.1 在商品管理页面添加导出按钮

修改 `meituan-frontend/src/views/Products.vue`：

在搜索栏区域添加"导出商品图片"按钮：

```vue
<el-button
  type="primary"
  @click="handleExportImages"
  style="background-color: #67C23A; border-color: #67C23A;"
>
  <el-icon><Download /></el-icon>
  导出商品图片
</el-button>
```

添加导出方法：

```javascript
import { exportProductImages } from '@/api/index.js'

const handleExportImages = async () => {
  try {
    // 显示命名方式选择对话框
    const { value: namingType } = await ElMessageBox.prompt(
      '请选择图片命名方式',
      '导出商品图片',
      {
        confirmButtonText: '确认导出',
        cancelButtonText: '取消',
        inputType: 'select',
        inputOptions: {
          'PRODUCT_NAME': '商品名称',
          'BARCODE': '条形码',
          'STORE_CODE': '店内码/货号'
        },
        inputValue: 'PRODUCT_NAME'
      }
    )

    const loading = ElLoading.service({
      lock: true,
      text: '正在导出图片，请稍候...',
      background: 'rgba(0, 0, 0, 0.7)'
    })

    try {
      const blob = await exportProductImages({
        merchantId: 1,
        namingType: namingType,
        productIds: selectedIds.value.length > 0 ? selectedIds.value : null
      })

      // 下载ZIP文件
      const url = window.URL.createObjectURL(blob)
      const link = document.createElement('a')
      link.href = url
      const timestamp = new Date().toISOString().replace(/[-:]/g, '').replace('T', '_').split('.')[0]
      link.download = `商品图片_${timestamp}.zip`
      document.body.appendChild(link)
      link.click()
      document.body.removeChild(link)
      window.URL.revokeObjectURL(url)

      ElMessage.success('图片导出成功')
    } finally {
      loading.close()
    }
  } catch (error) {
    if (error !== 'cancel') {
      ElMessage.error('图片导出失败')
    }
  }
}
```

## 测试步骤

### 1. 测试OSS连接

启动后端后，查看日志：
```
阿里云OSS客户端初始化成功
```

如果看到警告：
```
阿里云OSS配置未设置，图片上传功能将不可用
```

说明OSS配置有问题，请检查 `application.yml`。

### 2. 测试图片导出

1. 进入商品管理页面
2. 确保有商品数据且商品有图片
3. 点击"导出商品图片"按钮
4. 选择命名方式
5. 等待导出完成
6. 检查下载的ZIP文件

### 3. 验证导出结果

解压ZIP文件，检查：
- ✅ 图片命名是否正确
- ✅ 特殊字符是否已替换
- ✅ 图片大小是否小于5M
- ✅ 图片格式是否为JPG

## 故障排查

### 问题1：OSS初始化失败

**检查**：
- `application.yml` 中的OSS配置是否正确
- AccessKey是否有效
- Bucket是否存在
- 网络是否可以访问OSS

### 问题2：图片下载失败

**检查**：
- 图片URL是否可访问
- OSS Bucket权限是否为公共读
- 网络连接是否稳定

### 问题3：导出超时

**原因**：图片数量过多

**解决**：
- 分批导出（选择部分商品）
- 增加超时时间（前端API中的timeout参数）
