## 问题描述

## 解决方案

### 方案1：执行更新脚本（推荐）
如果数据库已经运行过迁移脚本，直接执行更新脚本：

```bash
mysql -u root -p meituan_product < update-admin-password.sql
```

或在数据库客户端中执行：

```sql
USE meituan_product;

UPDATE `user` 
SET `password` = '$2a$10$dXJ3SW6G7P50lGmMkkmwe.20cQQubK3.HZWzG3YB1tlRy.fqvM/BG'
WHERE `username` = 'admin';
```

### 方案2：重新运行迁移脚本
如果还没有运行迁移脚本，或者想重新初始化：

```bash
mysql -u root -p meituan_product < database-migration-member-management.sql
```
