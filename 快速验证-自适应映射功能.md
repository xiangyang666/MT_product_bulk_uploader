### ✅ 成功标志

1. **配置加载成功**
   - 日志显示"成功加载 30 个映射规则"

2. **v1.3.7模板识别**
   - "毛重" → weight字段（置信度0.95）
   - "毛重单位" → weightUnit字段

3. **v1.3.6模板兼容**
   - "重量" → weight字段（置信度1.0）

4. **映射性能**
   - 映射耗时 < 100ms

### ❌ 如果出现问题

#### 问题1：配置文件未加载
**日志显示：** "配置文件不存在，使用默认规则"

**解决方案：**
1. 检查文件是否存在：`meituan-backend/src/main/resources/mapping-rules.json`
2. 重新编译项目：`mvn clean compile`
3. 重启后端服务

#### 问题2：自适应映射未启用
**日志显示：** "使用固定映射（自适应映射已禁用或不可用）"

**解决方案：**
1. 检查 `application.yml`:
   ```yaml
   meituan:
     adaptive-mapping:
       enabled: true  # 确保为true
   ```
2. 重启后端服务

#### 问题3："毛重"列未映射
**日志显示：** "未找到列：'毛重'"

**解决方案：**
1. 检查 `mapping-rules.json` 中weight字段的配置
2. 确认 `exactMatches` 包含 "毛重"
3. 重启后端服务
