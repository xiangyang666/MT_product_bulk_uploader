# 商品图片功能 - 快速部署指南

## 一、数据库迁移（5分钟）

```bash
# 执行SQL脚本
mysql -u root -p < database-migration-product-images.sql
```

或手动执行：

```sql
USE meituan_product;
ALTER TABLE t_product ADD COLUMN product_images TEXT COMMENT '商品图片URL列表（JSON数组，最多5张）' AFTER image_url;
```

## 二、后端配置（10分钟）

### 1. 添加Maven依赖

编辑 `meituan-backend/pom.xml`，在 `<dependencies>` 标签内添加：

```xml
<!-- 图片处理库 -->
<dependency>
    <groupId>net.coobird</groupId>
    <artifactId>thumbnailator</artifactId>
    <version>0.4.19</version>
</dependency>

<!-- 阿里云OSS SDK -->
<dependency>
    <groupId>com.aliyun.oss</groupId>
    <artifactId>aliyun-sdk-oss</artifactId>
    <version>3.17.1</version>
</dependency>
```

### 2. 配置阿里云OSS

编辑 `meituan-backend/src/main/resources/application.yml`，添加：

```yaml
# 阿里云OSS配置
aliyun:
  oss:
    endpoint: oss-cn-hangzhou.aliyuncs.com
    access-key-id: 你的AccessKeyId
    access-key-secret: 你的AccessKeySecret
    bucket-name: 你的Bucket名称
    base-url: https://你的Bucket名称.oss-cn-hangzhou.aliyuncs.com
```

**获取OSS配置**：
1. 登录阿里云：https://oss.console.aliyun.com/
2. 创建Bucket（读写权限：公共读）
3. 获取AccessKey（用户中心 → AccessKey管理）

### 3. 编译后端

```bash
cd meituan-backend
mvn clean package -DskipTests
```

### 4. 重启后端服务

```bash
# 停止现有服务
# 启动新服务
java -jar target/meituan-backend-1.0.0.jar
```

## 三、前端部署（5分钟）

## 五、常见问题

### Q1: OSS初始化失败

**症状**：日志显示"阿里云OSS配置未设置"

**解决**：
1. 检查 `application.yml` 配置是否正确
2. 确认AccessKey是否有效
3. 确认Bucket是否存在

### Q2: 图片上传失败

**症状**：上传时报错

**解决**：
1. 检查图片大小（不超过10MB）
2. 检查Bucket权限（设置为"公共读"）
3. 检查网络连接

### Q3: 图片无法显示

**症状**：上传成功但无法显示

**解决**：
1. 设置Bucket为"公共读"权限
2. 配置CORS规则（允许所有来源）

## 六、功能说明

### 图片上传
- 每个商品最多5张图片
- 每张图片不超过10MB
- 支持jpg、png、jpeg等格式

### 图片导出
- 按美团规范导出
- 自动压缩超过5M的图片
- 支持三种命名方式
- 最多导出300张图片
