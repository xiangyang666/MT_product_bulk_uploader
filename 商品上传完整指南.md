# 商品上传完整指南

## 📋 完整上传流程（3步）

```
步骤1：导入商品数据（在本系统）
   ↓
步骤2：生成美团模板（在本系统）
   ↓
步骤3：上传到美团后台（在美团网站）
```

---

## 🎯 详细操作步骤

### 步骤 1：导入商品数据

#### 1.1 准备 Excel 文件

确保你的 Excel 文件包含以下信息：
- 商品名称
- 商品类目ID
- 价格
- 库存
- 商品描述
- 其他商品信息...

#### 1.2 导入到系统

1. **打开系统**
   - 访问：http://localhost:5173
   - 登录账号：admin / 123456

2. **进入批量导入页面**
   - 点击左侧菜单"批量导入"

3. **上传文件**
   - 点击"选择文件"按钮
   - 或直接拖拽 Excel 文件到上传区域

4. **预览数据**
   - 点击"预览"按钮
   - 检查数据是否正确
   - 系统会自动识别格式

5. **确认导入**
   - 点击"导入"按钮
   - 等待导入完成
   - 查看导入结果

✅ **完成后**：商品已保存到系统，状态显示"待上传"

---

### 步骤 2：生成美团模板

#### 2.1 进入批量上传页面

1. **点击左侧菜单"批量上传"**

2. **查看商品列表**
   - 系统会显示所有已导入的商品
   - 可以看到商品名称、类目ID、价格等信息

#### 2.2 选择要上传的商品

**方法 1：全选所有商品**
```
点击表格左上角的复选框 → 全选
```

**方法 2：选择部分商品**
```
逐个勾选商品前面的复选框
```

**方法 3：按条件筛选后选择**
```
使用搜索框筛选 → 勾选需要的商品
```

#### 2.3 生成模板

1. **点击"生成模板"按钮**
   - 按钮位置：页面右上角
   - 按钮颜色：美团黄色

2. **等待生成**
   - 系统会自动处理数据
   - 填充到美团官方格式
   - 生成 Excel 文件

3. **自动下载**
   - 文件会自动下载到你的电脑
   - 文件名格式：`meituan_template_20260209153045.xlsx`
   - 保存位置：浏览器默认下载文件夹

✅ **完成后**：你的电脑上有一个符合美团格式的 Excel 文件

---

### 步骤 3：上传到美团后台

#### 3.1 登录美团商家后台

1. **打开美团商家后台**
   - 网址：https://e.meituan.com/
   - 或搜索"美团商家后台"

2. **登录账号**
   - 输入你的美团商家账号
   - 输入密码
   - 完成登录

#### 3.2 进入商品管理

1. **找到商品管理入口**
   ```
   左侧菜单 → 商品 → 商品管理
   ```

2. **进入批量导入**
   ```
   商品管理页面 → 批量操作 → 批量导入
   或
   商品管理页面 → 导入商品
   ```

#### 3.3 上传文件

1. **选择文件**
   - 点击"选择文件"或"上传文件"
   - 找到刚才下载的 Excel 文件
   - 文件名：`meituan_template_xxxxxxxx.xlsx`

2. **确认上传**
   - 点击"确定"或"开始导入"
   - 等待美团处理

3. **查看结果**
   - 美团会显示导入进度
   - 显示成功/失败数量
   - 如有错误，会显示错误原因

✅ **完成后**：商品已上传到美团，可以在美团平台销售

## 💡 常见问题

### Q1：生成模板后，状态还是"待上传"？

**答**：正常的！状态要等你真正上传到美团后才会变。

**解决方法**：
- 系统无法自动知道你是否上传到美团
- 状态仅供参考，实际以美团后台为准

---

### Q2：找不到"生成模板"按钮？

**答**：需要先选择商品！

**解决方法**：
1. 勾选至少一个商品
2. "生成模板"按钮会变亮
3. 如果没选商品，按钮是灰色的

---

### Q3：下载的文件在哪里？

**答**：在浏览器的默认下载文件夹。

**查找方法**：
- **Chrome**：右上角 → 下载内容
- **Edge**：右上角 → 下载
- **或者**：打开"此电脑" → "下载"文件夹

---

### Q4：美团上传失败怎么办？

**答**：检查数据是否完整。

**常见原因**：
1. ❌ 商品名称为空
2. ❌ 类目ID不存在
3. ❌ 价格格式不对
4. ❌ 必填字段缺失

**解决方法**：
1. 查看美团的错误提示
2. 修正原始 Excel 数据
3. 重新导入到系统
4. 重新生成模板
5. 再次上传到美团

---

### Q5：可以只上传部分商品吗？

**答**：可以！

**操作方法**：
1. 在"批量上传"页面
2. 只勾选想上传的商品
3. 点击"生成模板"
4. 只会生成选中商品的模板

---
