# 🚀 批量导入问题 - 快速解决指南

## 问题
上传3752行商品数据时出现"类目ID必须是10位数字"错误

## 原因
美团导出的数据格式与系统要求不匹配

## 解决方案（3步搞定）

### 第1步：重启后端 ⚙️
```bash
start-backend.bat
```

### 第2步：转换数据 🔄
1. 双击打开 `数据转换工具.html`
2. 上传您的美团导出文件（3752行）
3. 查看转换结果
4. 点击"下载转换后的文件"

### 第3步：导入系统 📥
1. 打开应用，进入"批量导入"页面
2. 上传转换后的文件
3. 预览数据
4. 确认导入

## 工具特点
✅ 自动识别美团数据格式  
✅ 自动验证类目ID（10位数字）  
✅ 显示详细错误报告  
✅ 一键生成标准导入文件
