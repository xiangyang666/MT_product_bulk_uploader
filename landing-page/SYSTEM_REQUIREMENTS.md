# 美团商品批量上传系统 - 系统要求

## 最低系统要求

### 操作系统
- Windows 10 (64位) 或更高版本
- Windows 11 (推荐)

### 硬件要求
- **处理器**: Intel Core i3 或同等 AMD 处理器
- **内存**: 4GB RAM (推荐 8GB)
- **硬盘空间**: 至少 200MB 可用空间
- **显示器**: 1280x720 分辨率或更高

### 软件依赖
- Java Runtime Environment (JRE) 8 或更高版本
- MySQL 5.7 或更高版本（用于数据存储）
- 稳定的网络连接（用于连接美团 API）

## 推荐配置

### 操作系统
- Windows 11 (64位)

### 硬件要求
- **处理器**: Intel Core i5 或更高
- **内存**: 8GB RAM 或更高
- **硬盘空间**: 1GB 可用空间
- **显示器**: 1920x1080 分辨率

### 网络要求
- 宽带网络连接
- 上传速度至少 2Mbps（用于图片上传）

## 浏览器要求（Web 界面）

如果使用 Web 界面访问：
- Google Chrome 90+ (推荐)
- Microsoft Edge 90+
- Firefox 88+

## 注意事项

1. 首次安装需要管理员权限
2. 建议关闭防火墙或添加程序到白名单
3. 确保 MySQL 服务正常运行
4. 需要有效的美团商家账号和 API 密钥

## 安装步骤

1. 下载安装包
2. 双击运行安装程序
3. 按照安装向导完成安装
4. 配置数据库连接
5. 输入美团 API 密钥
6. 开始使用
