# 按钮样式优化 - 无边框

## ✅ 修改内容

### 折叠和刷新按钮样式优化

**修改前：**
```css
.action-btn {
  border: 1px solid #e0e0e0;  /* 有边框 */
  background-color: #fff;      /* 白色背景 */
}

.action-btn:hover {
  background-color: #FFD100;
  border-color: #FFD100;       /* 悬停时边框变黄 */
}
```

**修改后：**
```css
.action-btn {
  border: none;                /* 无边框 ✅ */
  background-color: transparent; /* 透明背景 ✅ */
}

.action-btn:hover {
  background-color: #FFD100;
  border-color: transparent;   /* 悬停时也无边框 ✅ */
}
```

---

## 🎨 视觉效果

### 默认状态
- ❌ 修改前：白色背景 + 灰色边框
- ✅ 修改后：透明背景 + 无边框

### 悬停状态
- ✅ 美团黄色背景
- ✅ 深色图标
- ✅ 无边框

---

## 💡 优势

### 1. 更简洁
- 去除了多余的边框
- 视觉更清爽
- 更符合现代设计风格

### 2. 更融合
- 透明背景与导航栏融为一体
- 不会显得突兀
- 只在悬停时显示背景色

### 3. 更专业
- 类似macOS的设计风格
- 简约而不简单
- 提升整体美感
